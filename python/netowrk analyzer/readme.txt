Program SmartClient is used to analyze network connection protocols and determine whether they support HTTP2, password protection etc

How to use PMan
Step 1: compile test files that you want to use
Step 2: execute ./python3 SmartClient.py "url of users choice" to launch the program (omit quotations)


Important notes for SmartClient:
-if an error is encountered with URL, try it without "https://www"
