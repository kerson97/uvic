Program capScanner is used to analyze TCP connection details by parsing the captured .pcap file

How to use capScanner
Step 1: compile test files that you want to use
Step 2: execute ./python3 pcapScanner "example.cap" to launch the program (omit quotations, chosen cap must be in same directory as capScanner)
